# Configuration

## Files

- `config/example-dht22.yaml` : working example configuration (recommended starting point)
- `secrets.yaml` : Wi-Fi + OTA/API secrets (kept out of git)

## Pins
Default DATA pin is `GPIO4`. Change in the YAML if needed.

## Static IP (optional)
Uncomment the `manual_ip:` section if you want a static IP address.
